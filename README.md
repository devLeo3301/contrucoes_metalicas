## 💎 Diferenciais do Design

- **Estética Industrial Premium:** Paleta de cores baseada em azul profundo e acentos laranjas para evocar confiança e vigor técnico.
- **Micro-interações:** Navegação inteligente com detecção de scroll, efeitos de brilho (shimmer) em botões de conversão e animações de entrada suaves.
- **Portfolio Dinâmico:** Sistema de filtragem de projetos em tempo real para uma navegação fluida entre categorias de obras.
- **Triagem Automatizada:** Formulário de contato inteligente que formata as necessidades do cliente e as envia diretamente para o WhatsApp do responsável técnico.

---

## 🚀 Tecnologias

Este projeto utiliza o estado da arte do ecossistema React:

- **Next.js 15 (App Router):** Roteamento avançado e performance otimizada.
- **Tailwind CSS v4:** A nova geração do framework CSS, utilizando configuração nativa via CSS variables (Zero-config).
- **Lucide React:** Conjunto de ícones minimalistas e consistentes.
- **Framer Motion:** Animações fluidas e declarativas.
- **Unsplash API integration:** Imagens de alta resolução dinâmicas para portfólio.

## 🛠️ Funcionalidades principais

### 🏠 Home
- Hero Section de alto impacto com tipografia robusta.
- Banner de autoridade com estatísticas de mercado.
- Seção "Sobre" com foco em diferenciais técnicos.

### ⚙️ Serviços
- Apresentação detalhada com layout intercalado.

### 📸 Portfólio
- Grid interativo com efeito de zoom no hover.

### 📱 Contato & Conversão
- Validação de campos obrigatórios.
- Redirecionamento inteligente via `wa.me` com mensagens pré-formatadas em Markdown para o WhatsApp.
